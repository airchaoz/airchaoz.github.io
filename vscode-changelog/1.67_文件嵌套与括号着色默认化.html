核心亮点

01

Explorer 文件嵌套

基于文件名自动将生成的文件嵌套在其源文件下方，支持自定义模式，复制/拖拽/删除时折叠操作视为一组。

02

设置编辑器过滤器按钮

设置搜索栏新增漏斗按钮，可快速应用 @Modified 等筛选条件，精准定位变更项。

03

括号对颜色化默认启用

自 1.60 引入后广受好评，现默认开启，可通过 editor.bracketPairColorization.enabled 关闭。

04

切换内联提示

新增 onUnlessPressed / offUnlessPressed 模式，按住 Ctrl+Alt 快速显隐参数名等内联提示。

05

拖放创建 Markdown 链接

将文件从资源管理器或外部拖入 Markdown 编辑器，自动生成链接或图片语法。

06

Markdown 查找所有引用

支持对标题、引用链接、文件、URL 执行 Find All References，跨文件定位。
