核心亮点

01

标题栏右键定制

右键标题栏可快速隐藏/显示菜单栏、命令中心与布局控件，简化UI个性化操作。

02

手动折叠区域

通过命令从选中行创建自定义折叠范围，并支持保留折叠状态，即使语法变化也不丢失。

03

搜索多选批量操作

搜索视图支持多选结果，可一次性执行替换、删除等操作，大幅提升重构效率。

04

终端Shell集成默认开启

Shell集成默认激活，自动显示命令装饰、支持运行最近命令、扩展PowerShell键绑定，终端体验脱胎换骨。

05

命令行3-way合并

新增 `--merge` 选项，可直接在VS Code中启动三路合并编辑器，并可作为Git默认合并工具。

06

编辑器粘性滚动预览

滚动时顶部固定显示当前所在类/函数等作用域，帮助大型文件中快速定位代码上下文。

Editor

工作台与编辑器

  • 树视图查找控件：文件资源管理器等树视图支持 ⌥⌘F 弹出查找框，可高亮或过滤匹配项。
  • 搜索文件装饰：搜索结果视图现在显示文件问题与源代码管理图标，与文件资源管理器一致。
  • 新建文件支持任意路径：通过文件 > 新建文件... 可直接输入路径创建深层目录文件，无需先建文件夹。
  • 三路合并编辑器默认启用：冲突文件点击后自动打开新的合并编辑器，支持一键接受全部更改、改进的差异颜色。
  • JSON格式化保留换行：新增 JSON > Format: Keep Lines 设置，格式化时保留数组元素原有行位置。
Terminal

终端

  • Shell集成自动注入：bash/zsh/pwsh 自动开启命令装饰，支持 Ctrl+Space 等扩展键绑定，无需手动配置。
  • 简化手动安装脚本：通过 code --locate-shell-integration-path 一行命令即可在远程或 dotfiles 中安装。
  • 命令装饰可配置显示位置：新设置 terminal.integrated.shellIntegration.decorationsEnabled 控制装饰显示在侧边栏或概览标尺。
  • 默认终端标签图标与颜色：可为每个文件夹设置默认的终端标签图标（terminal.integrated.tabs.defaultIcon）和颜色。
  • 多行命令粘贴支持：Shell集成下多行命令会以粘贴模式一次性发送到同一提示符，避免错误执行。

其他特性

  • JS调试终端参数传递改进：支持 argsCanBeInterpretedByShell，用户可选择是否让Shell解释特殊字符。
