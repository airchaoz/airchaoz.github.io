核心亮点

01

本地历史记录 (Local History)

在 Timeline 视图中自动保存文件历史，支持对比、恢复、重命名，无需依赖版本控制。

02

终端查找高亮与滚动条标注

搜索关键词在终端面板和滚动条中高亮显示，快速定位匹配结果。

03

内置 CSS/LESS/SCSS 格式化器

无需安装额外扩展即可对样式文件进行格式化，支持多种配置项。

04

JavaScript 堆分析 (Heap Profiles)

调试时可采集并可视化内存分配堆快照，帮助定位内存泄漏。

05

SSH 远程支持 Apple Silicon/M1/ARM64

Remote - SSH 扩展可连接到 Apple Silicon 或 ARM64 远程机器。

06

VS Code for Web 拖放文件夹

在浏览器中直接拖拽本地文件夹到 vscode.dev 打开，提升在线开发体验。

重点功能

重点功能速览

  • 编辑器：设置编辑器新增 @lang:languageId 语言过滤器，可快速查看和修改语言特定设置。工作区和文件夹设置现在会保留直到手动重置。快速建议（editor.quickSuggestions）支持 inline 值，将补全显示为灰显文本。括号对颜色化新增 editor.bracketPairColorization.independentColorPoolPerBracketType 设置，不同类型括号使用独立颜色池。Markdown 简写引用链接（如 [my fancy link]）现在可在编辑器中点击跳转。
  • 终端：终端查找结果在滚动条中显示标注。命令行导航（Ctrl+Up/Down）在启用 shell integration 时更可靠，并显示 UI 反馈。新增 Terminal: Copy Selection as HTML 命令保留富文本样式。最小对比度默认值改为 4.5（WCAG AA），提升可访问性。
  • 源代码管理：源代码管理存储库视图现在按名称排序，而非发现顺序。文件修改装饰器增加了图案和高对比度，提升可见性。
  • 调试：JavaScript 调试器支持收集和可视化堆配置文件，通过 Debug: Take Performance Profile 命令或调用堆栈视图的录制按钮访问。
  • 语言支持：TypeScript 4.6.3 捆绑。Markdown 简写引用链接可点击。新增 reStructuredText (rst) 文件语法高亮内置扩展。基于历史的语言检测现在默认启用，并新增 workbench.editor.preferHistoryBasedLanguageDetection 提高优先级。
  • Web 版与远程开发：VS Code for Web 支持拖放文件夹。Azure Repos 扩展重构为依赖 Remote Repositories 公共扩展，支持编辑提交和 PR 创建。Remote - SSH 支持 Apple Silicon/M1/ARM64。
  • UI 与设置：新增 workbench.localHistory 系列设置控制本地历史行为。通知的第一个按钮显示为主题主色。新的上下文键 activeEditorIsFirstInGroup 和 activeEditorIsLastInGroup 可用于键盘快捷键。workbench.editor.defaultBinaryEditor 设置可绕过二进制文件警告直接打开指定编辑器。音频提示新增音量控制（audioCues.volume）。减少动画模式可通过 workbench.reduceMotion 设置。侧边栏和侧面板重命名为“主侧边栏”和“辅助侧边栏”。
