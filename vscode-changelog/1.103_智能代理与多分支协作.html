CHAT

Chat & AI

  • 终端自动审批改进：合并 allowList/denyList 为新设置，支持正则标志（如 /i 忽略大小写），匹配可通过 matchCommandLine 作用于完整命令行。
  • 任务/终端输入请求检测：Agent 模式可检测脚本等待用户输入，并在聊天中弹出确认提示（默认选项高亮）。
  • 改进的错误检测与问题匹配器：任务使用问题匹配器时，Agent 直接提取错误结果而非依赖模型判断，错误以下拉形式展示并可导航。
  • 复合任务支持：Agent 模式可运行复合任务，显示每个依赖任务的进度和输出。
  • 实验性任务列表（Todo List）：Agent 自动将高层次任务拆解为子任务列表，实时更新完成状态，需启用 chat.todoListTool.enabled。
  • 编辑历史请求：默认点击请求气泡即可原地编辑，支持修改附加项、模式和模型并重新发送。
  • 数学公式渲染（预览）：聊天支持 KaTeX 渲染行内/块级数学公式，默认关闭，可通过 chat.math.enabled 开启。
  • MCP 服务器自动启动与信任对话框：新增 chat.mcp.autostart 设置，服务器首次启动时弹出信任确认。
SOURCE

Source Control & Terminal

  • Git 工作树增强：在源代码管理仓库视图区分仓库、子模块和工作树，并显示父子关系。
  • 终端内置文档建议：基于 LSP 的终端建议现包含内联文档（如 Python REPL），需启用相关设置。
  • 终端语音听写：重新引入终端语音输入，支持 Gemini 和 Claude 扩展的终端自然语言输入。
  • 改进的 Shell 集成诊断：悬停终端时显示检测到的 shell 类型和当前工作目录，便于排查问题。
  • 终端内联聊天改进：动态检测当前 shell（包括子 shell），提供更准确的命令建议。
