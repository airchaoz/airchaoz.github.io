核心亮点

01

Agent 插件系统

安装预打包的技能、工具和钩子集合，从扩展视图搜索 @agentPlugins 即可发现，无需手动配置多文件。

02

代理浏览器工具

让代理自主驱动集成浏览器，执行点击、输入、截图等操作并验证页面错误，实现开发与验证闭环。

03

Agent 调试面板

实时查看代理事件、系统提示、工具调用与自定义加载情况，以层级图呈现，替换旧诊断功能。

04

会话记忆与上下文压缩

计划自动持久化并跨轮次保留，聊天接近上限时可手动 /compact 压缩，避免丢失重要上下文。

05

分叉会话

通过 /fork 或右键从任意检查点创建独立新会话，继承历史探索不同方向，互不影响。

06

终端支持 Kitty 图形协议

直接在终端渲染 PNG/RGB 图像，支持缩放、裁剪、z-index 和压缩传输，需启用 enableImages 设置。

智能代理

智能代理与 Copilot Chat

  • 背景代理与 Claude 代理增强：背景代理支持后台执行任务并同步状态；Claude 代理新增转向、排队、上下文窗口渲染和 /compact 命令。
  • 创建自定义化从聊天：使用 /create-prompt、/create-skill 等命令直接从对话生成 prompt、instruction、skill 文件，也可用自然语言触发。
  • 自动批准与转向控制：通过 /autoApprove 命令全局启用工具自动批准，加速多步任务；转向消息可在不中断流式响应时发送。
  • OS 通知与内联聊天队列：可配置始终显示 OS 通知（即使窗口聚焦）；内联聊天自动排队到同一会话，确保完整上下文。
Editor

编辑器体验

  • 模态编辑器（实验性）：设置 workbench.editor.useModal 后，设置、快捷键等编辑器以浮动模态打开，不影响标签栏布局。
  • 可配置通知位置：新增 workbench.notifications.position 设置，可在右下、右上、左下中选择，避免遮挡聊天视图。
  • 远程未来编辑建议（NES）增强：长距离 NES 可预测文件任意位置的后续编辑，NES 状态栏新增急迫度选项控制建议频率与相关性。
  • 内联聊天悬浮 UI 与提示：内联聊天可选悬浮模式（类似重命名体验），选中代码后可在编辑器内或行号区显示入口菜单。
