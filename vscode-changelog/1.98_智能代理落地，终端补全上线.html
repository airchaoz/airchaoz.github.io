GITHUB

GitHub Copilot

  • Agent mode 增强：终端命令内联显示、可编辑后再运行；支持自动运行 build 任务；可逐步骤撤销文件修改。
  • Copilot Edits 支持 Notebook：可在 Notebook 中创建、修改、插入/删除单元格，编辑体验与代码文件一致。
  • Copilot Vision 图像交互：拖拽或粘贴图片到 Chat，支持截图窗口，当前仅 GPT 4o 支持。
  • #codebase 高级搜索：可运行文本搜索、文件搜索、语义搜索等工具，为 Chat 提供更精准的上下文。
  • 自定义指令正式可用：workspace 中 .github/copilot-instructions.md 可覆盖默认行为，需启用 setting。
  • 折叠模式与模型切换：Next Edit Suggestions 支持折叠模式（仅显示边距指示器）；可切换补全模型。
  • 问题面板附加到 Chat：可拖拽或输入 #problems 将诊断问题作为上下文发送给 Copilot。
  • 文件夹附加与权限提示优化：通过 paperclip 附加文件夹；GitHub 仓库鉴权改用非模态确认框，支持“不再询问”。
  • Copilot 状态概览 (Experimental)：状态栏 Copilot 图标显示 quota、编辑器设置及快捷键概览。
Terminal

终端与调试

  • 终端 IntelliSense 预览：启用后提供 shell 命令补全，新增 30+ CLI 补全规范（git、npm、brew 等）。
  • 增强的补全细节：bash/zsh/PowerShell 内置命令显示参数描述；支持 CDPATH、别名、绝对路径。
  • 内联建议检测与排序：准确识别 shell 级自动建议并置顶显示；排序优化（带详情的靠前，路径前缀点降权）。
  • 快速建议配置：新增 terminal.integrated.suggest.quickSuggestions 设置，可按命令/参数/未知类别控制。
  • 调试内联值悬停：启用 debug.inlineValues 后，内联值装饰支持悬停查看完整内容。
Editor

编辑器与源码管理

  • Peek References 拖拽支持：可从 Peek 视图拖拽条目到新编辑器，快速打开引用。
  • Linux 自定义标题栏默认启用：可使用布局控件、Copilot 菜单，可通过 window.titleBarStyle 回退。
  • 高亮出现延迟归零：editor.occurrencesHighlightDelay 默认 0，编辑时更即时。
  • 软删除未跟踪文件：git.discardUntrackedChangesToTrash 默认启用，放弃时移至回收站。
  • 诊断提交钩子 (Experimental)：提交时若有未解决的诊断错误则提示，可自定义诊断源和级别。
  • 源码管理视图标题精简：“Repositories”“Changes”“Graph” 替代冗长标题。
  • Notebook 内联 diff 视图 (Experimental)：可在 Notebook 差异对比中使用内联视图替代并排对比。
