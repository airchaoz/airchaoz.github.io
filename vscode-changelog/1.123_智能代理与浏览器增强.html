核心亮点

01

Agent 窗口支持多会话并排

可在专用窗口中同时打开多个代理会话，支持拖拽、Alt+点击等快速并排，方便对比或审查不同工作成果。

02

研究代理（Preview）

输入 /research 即可对代码库、GitHub 和网络进行深度研究，生成带有引用的 Markdown 报告，帮助理解复杂代码或比较方案。

03

1M Token 上下文窗口

Anthropic 和 OpenAI 模型支持百万级 token 上下文，可处理更庞大的代码仓库和长对话，减少上下文丢失。

04

会话同步与 Chronicle

聊天会话自动同步到 GitHub 账户，支持自然语言查询历史、生成站会报告，并可通过 /chronicle 命令检索过往上下文。

05

集成浏览器：收藏夹与区域截图

URL 栏支持添加收藏页，并可从浏览器直接截取矩形区域或整页截图作为聊天上下文，提升 UI 调试效率。
