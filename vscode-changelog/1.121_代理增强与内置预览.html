核心亮点

01

远程代理（Preview）

可通过 SSH 或 Dev Tunnels 在远程机器上运行代理会话，断开客户端连接后任务继续执行，并支持多客户端同时协调。

02

Mermaid 图内置渲染

Mermaid Markdown Features 成为内置扩展，在 Markdown 预览、Notebook 及聊天中直接渲染 Mermaid 图，支持缩放和右键复制源码。

03

HTML 文件原生预览

无需安装扩展，在文件资源管理器或编辑器标签中右键即可用集成浏览器打开本地 HTML 文件。

04

终端工具输出压缩与自动清理

对测试运行器、构建工具等输出进行深度压缩以节省 tokens，并在命令完成后自动回收后台终端，减少资源占用。

05

实用模型可自定义配置

新增 chat.utilityModel 和 chat.utilitySmallModel 设置，允许用户覆盖生成提交消息、摘要等轻量任务使用的模型，支持 BYOK。

06

快速建议默认关闭以减少干扰

当存在内联补全提供者（如 Copilot）时，默认不再自动弹出建议控件，避免误选全局符号导致建议噪声。

重点功能

重点功能速览

  • Agents：代理窗口（Preview）持续改进，远程代理支持 SSH/Dev Tunnels 连接，Claude Agent 引入 Auto 权限模式（自动执行但保留安全审查），终端敏感提示（密码、PIN 等）被拦截确认或取消命令。
  • Language Models：新增 chat.utilityModel 和 chat.utilitySmallModel 配置实用模型；Insiders 版增加 Custom Endpoint provider，兼容 Chat Completions、Responses、Messages 三种 API 家族。
  • Integrated Browser：HTML 文件可通过右键菜单或编辑器标题栏预览图标直接打开；元素选择 UI 支持拖拽范围选择和右键快速附加到聊天。
  • Terminal：为代理启动的终端设置 VSCODE_AGENT 环境变量；工具调用在后台运行时显示“Running in background”指示并可展开查看；命令结束后自动清理背景终端；扩展输出压缩覆盖 pytest、jest、cargo test、tsc 等。
  • Editor：editor.quickSuggestions 默认值变更：当存在内联补全提供者时，输入字母不再自动触发建议控件，可回退旧行为。
  • Languages：Mermaid 图内置渲染；Markdown 预览新增 markdown.preview.frontMatter 设置，支持以表格、代码块或隐藏方式显示 YAML frontmatter。
