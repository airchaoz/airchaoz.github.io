Security

工作台与安全

  • Workspace Trust：通过 Restricted Mode 自动禁用任务、调试、工作区设置和扩展，降低恶意代码风险
  • 全新欢迎页 (Getting Started)：默认启用新欢迎体验，支持扩展贡献引导教程，未完成时显示更多近期工作区
  • 编辑器拖拽改进：支持将 diff 编辑器、自定义编辑器和整个编辑器组拖拽到其他窗口
  • 移除 Cmd+W 关闭窗口快捷键：当无编辑器打开时不再默认关闭窗口，防止误操作；可手动恢复绑定
  • Quick Pick 和建议部件颜色更新：新增 focus 前景色 token，对齐树控件风格
  • macOS Touch Bar 图标更新：统一图标风格
  • Webview 右键菜单：在 Markdown 预览、发行说明等 webview 中支持右键复制粘贴
Editor

编辑器与语言

  • Suggestion Preview (editor.suggest.preview)：启用后，当选中建议时在光标位置实时预览建议内容
  • 隐藏废弃建议 (editor.suggest.showDeprecated)：关闭后可过滤掉标记为废弃的代码补全项
  • TypeScript 4.3.2：集成新版本 TypeScript，带来多项语言改进
  • JSDoc @link 导航：在悬停、补全和签名帮助中支持 @link 符号跳转
  • Import 语句自动补全：在编写 import 语句时自动提供模块内符号的补全
  • Go to Definition 支持非 JS/TS 文件：支持图片、样式表等资源文件的跳转定义
  • Sort Imports 代码操作：对 import 排序但不移除未使用项，可配置保存时自动执行
  • 推断函数返回类型快速修复：为 TypeScript 函数添加显式 return type 注释
