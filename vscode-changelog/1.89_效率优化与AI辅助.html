核心亮点

01

Markdown 图片/视频悬停预览

无需打开 Markdown 预览，悬停链接即可直接查看图片和视频，减少上下文切换。

02

切换分支时保存/恢复编辑器状态

切换 Git 分支时自动保存并恢复所有打开的编辑器，解决频繁切换分支丢失工作区的问题（需启用 scm.workingSets.enabled）。

03

终端中键粘贴

在终端中用鼠标中键直接粘贴剪贴板内容，Linux 用户尤为习惯，现已跨平台支持（terminal.integrated.middleClickBehavior 设为 paste）。

04

Quick Search 正式可用

通过命令面板快速搜索工作区文件，不再为实验性功能，极大提升文件定位速度。

05

右键直接自定义 UI 快捷键

在任何 UI 动作上右键选择「自定义快捷键」，自动携带 when 子句，简化键位绑定流程。

编辑器与语言

  • Markdown 路径补全预览：插入图片/视频路径时，补全详情中显示小预览图，帮助快速选择正确文件。
  • Markdown 标题重命名改进：重命名重复标题时，自动更新所有相关的锚点链接，避免死链。
  • 自动检测系统颜色模式改进：主题选择器现在只显示与当前系统模式匹配的主题（暗/亮），并新增直达设置按钮。
  • 自定义编辑器标签支持 Quick Open：此前为标签设置的自定义标题现在也出现在 Quick Open（Cmd+P）结果中，文件导航更直观。
  • 笔记本最小错误渲染器：使用 setting notebook.output.minimalErrorRendering 只显示错误消息和展开堆栈的控件，减少输出干扰。

  • Git Bash shell 集成默认启用：Git Bash 自动获得命令导航、sticky scroll、快速修复等终端集成功能，无需手动配置。

  • Canvas 渲染器弃用预告：Canvas 渲染器已从降级链移除，下个版本彻底删除；推荐使用 DOM 或 WebGL 渲染器。
  • JavaScript Debugger 自动查找 node_modules/.bin：launch.json 中 runtimeExecutable 指定名称（如 mocha）即可自动解析路径，无需写完整路径。
  • 十六进制编辑器插入模式：新增插入模式（按 Insert 键切换），可在文件中间或末尾插入新字节，不再只能替换。
