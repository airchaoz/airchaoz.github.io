启用后 agent 自动批准工具调用、重试错误、回复问题，持续迭代直到任务完成，无需人工干预。

Agent 作用域钩子预览：自定义预处理与后处理逻辑

在特定 agent 的 YAML frontmatter 中定义 hooks，实现仅在选定 agent 或 runSubagent 时触发的逻辑，互不影响。

调试事件快照：一键排查 agent 行为

通过 #debugEventsSnapshot 将 agent 调试事件附加到聊天中，快速了解自定义加载、令牌消耗等问题。

05

聊天提示体验重构：结构化引导新功能

按步骤展示基础提示（如 Plan agent 使用）与效率提示，自动隐藏多会话时的冗余提示，并集成快捷键。
