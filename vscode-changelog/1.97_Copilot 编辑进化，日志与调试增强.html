核心亮点

01

Copilot Next Edit Suggestions (NES) 预览

Copilot 不仅补全当前输入，还能预测下一个需要修改的位置并给出编辑建议，Tab 即可导航和接受，极致提升编辑效率。

02

Copilot Edits 正式发布

跨文件编辑能力从预览转为 GA，支持逐个接受/丢弃编辑，并新增自动接受超时设置 (chat.editing.autoAcceptDelay)，无需手动确认。

03

命令面板与 Quick Input 可拖动重新定位

告别固定在窗口顶部，现在可以拖拽命令面板到任意位置（吸附至中央或顶部），位置跨窗口持久保留。

04

扩展发布者信任机制

首次安装来自某发布者的扩展时会弹出信任评估对话框，确保开发环境安全；已安装的发布者自动受信任。

05

输出面板过滤与复合日志

输出面板支持按日志级别、分类、文本过滤；可组合多个日志为单一视图，还能导出/导入日志，极大简化日志分析。

06

Git Blame 信息增强与 GitHub 集成

编辑器内和状态栏的 Blame 信息默认启用，悬停可查看更丰富的详情，并新增“在 GitHub 上打开”命令，Issue/PR 引用直接渲染为链接。
