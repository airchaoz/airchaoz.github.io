核心亮点

01

远程隧道预览

无需 SSH 即可通过 vscode.dev 或 Remote - Tunnels 扩展安全连接到任何机器，实现远程开发。

02

管理不安全 Git 仓库

VS Code 检测非用户拥有文件夹的仓库并阻止 Git 操作，可通过命令标记为安全。

03

JavaScript console.profile 支持

调试时收集 CPU profile 并保存为 .cpuprofile 文件，可在 VS Code 内直接查看。

04

从 return 跳转到定义

在 JS/TS 中点击 return 关键字可快速跳转到函数顶部，方便导航嵌套函数。

05

音频提示与合并编辑器改进

笔记本执行成功/失败、差异编辑器插入/删除行均有独立音效；合并编辑器支持撤销/重做冲突操作。

Editor

工作台与编辑器

  • Explorer 自动显示排除：新设置 explorer.autoRevealExclude 使用 glob 模式排除特定文件/文件夹（如 node_modules）。
  • 隐藏活动栏和面板徽章：右键视图容器可隐藏其徽章（数字、图标等），简化 UI。
  • CJK 字符换行控制：editor.wordBreak 设为 keepAll 时避免在 CJK 字符后自动换行。
  • OpenType 字体变量支持：editor.fontVariations 允许直接配置 wght 等轴，实现更精确的字体粗细。
  • 缩进大小与 Tab 大小解耦：editor.indentSize 独立于 editor.tabSize，便于处理混用空格和 Tab 的文件。
  • 部分接受内联建议：命令 Accept Next Word Of Inline Suggestion 可逐个词接受代码补全内容。
  • TypeScript 4.9 内置：带来 satisfies 操作符、auto accessors 等新语言特性及文件监控改进。
SCM

源代码管理与 Git

  • 乐观 UI 更新：暂存、取消暂存、丢弃更改和提交后立即更新源代码管理视图，无需等待 git status 完成。
  • 提交按钮动作反馈：执行同步等二级命令时按钮标签动态变化并显示进度动画。
  • 保护分支锁图标：VS Code for the Web 在状态栏和分支选择器中为保护分支显示锁图标。
  • Git 设置新增 Web 支持：git.branchRandomName.enabled 等设置现在可用于 GitHub/Azure Repos 仓库的 Web 编辑。
Debug

远程开发与调试

  • 远程隧道 (Preview)：可通过 Turn on Remote Tunnel Access 或下载 CLI 建立隧道，从任何设备安全连接。
  • 调试：killOnServerStop：serverReadyAction 新增 killOnServerStop 属性，停止服务器调试时自动关闭浏览器调试会话。
