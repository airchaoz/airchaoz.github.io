核心亮点

01

配置文件徽章与快速切换

当前自定义配置文件名显示为活动栏徽章，并通过命令面板 (Profiles: Switch Profile) 快速切换，远程工作区也支持配置文件。

02

音频提示与终端屏幕阅读器改进

命令失败、行错误等场景新增音频提示；终端可访问缓冲区支持选择和链接，只读模式并借助 shell 集成过滤输出。

03

可移动的资源管理器视图

资源管理器视图容器可拖拽至辅助侧边栏或底部面板，配合重置布局命令实现高度自定义工作台。

04

笔记本内核 MRU 列表

内核选择器默认显示最近使用的内核，通过『选择其他内核…』可查看所有内核并按来源分组。

05

Markdown 工作区标头链接补全

在 Markdown 链接中键入 ## 即可从当前工作区所有文件中补全标头，由 markdown.suggest.paths.includeWorkspaceHeaderCompletions 控制。

  • 固定宽度居中布局：新增 workbench.editor.centeredLayoutFixedWidth 设置，居中布局下编辑器宽度在窗口/面板调整时保持不变。
  • 多个快速差异来源选择：当多个扩展提供快速差异时，编辑器内显示下拉菜单选择 diff 基准。

  • 独立括号对匹配与着色：editor.language.brackets 和 editor.language.colorizedBracketPairs 现在取并集用于匹配，仅着色后者配置的括号。
