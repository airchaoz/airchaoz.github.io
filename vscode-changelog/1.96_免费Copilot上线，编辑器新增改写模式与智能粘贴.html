核心亮点

01

GitHub Copilot Free 计划

现在无需订阅即可每月获得一定数量的代码补全与聊天交互，只需 GitHub 账号即可在 VS Code 内激活使用。

02

编辑器改写模式 (Overtype)

支持用 Insert 键或命令切换改写/插入模式，状态栏显示 OVR 指示，方便对齐 Markdown 表格等场景。

03

粘贴时自动添加导入 (Paste with imports)

在 JavaScript/TypeScript 文件间复制代码时，VS Code 可自动补充缺失的 import 语句并通过粘贴控件选择是否应用。

04

终端合字支持 (Terminal Ligatures)

无论是否启用 GPU 加速，现在终端均可显示合字，需要配合支持合字的字体使用。

05

可归因测试覆盖率 (Attributable Coverage)

扩展可提供每个测试所覆盖的代码，通过过滤器按钮快速查看特定测试覆盖了哪些行。

06

Copilot 调试助手 (copilot-debug)

在终端命令前加上 copilot-debug 即可自动配置 launch.json 并启动调试会话，支持预编译任务。

Editor

编辑器

  • 粘贴/拖放行为配置：新增 editor.pasteAs.preferences 和 editor.dropIntoEditor.preferences 设置，可指定默认粘贴方式（如绝对路径、带导入文本），并支持快捷键绑定。Find 控件的搜索历史现在可跨会话持久化，按工作空间存储。
  • TypeScript 5.7 支持：升级至 TypeScript 5.7，包含语言与工具改进，且粘贴导入功能仅限 TS 5.7+。
  • Experimental Git Blame：通过 git.blame.editorDecoration.enabled 启用编辑器内 Git Blame 装饰，可自定义显示模板（提交主题、作者、日期）。
GITHUB

GitHub Copilot

  • Copilot Edits 增强：流式编辑进度可视化、工作集建议文件、重启后恢复编辑会话、可从资源管理器/搜索/编辑器添加上下文文件。
  • 聊天上下文增强：支持从大纲视图或面包屑拖拽符号作为上下文，输入 # 后自动补全符号名，也可拖拽文件夹作为上下文。
  • 内联聊天改进：进度报告更轻量，流式变更时禁用波浪线，支持伪代码触发、空行提示（实验性）。终端内联聊天 UI 重做，更贴近编辑器内联聊天。
  • 自定义提交消息指令：可通过设置 github.copilot.chat.commitMessageGeneration.instructions 指定提交消息格式。
  • @workspace 性能优化：针对分支和 PR 场景，合并 GitHub 代码搜索与本地文件索引，提速上下文检索。
Terminal

终端与工作台

  • 终端合字与新变量：terminal.integrated.fontLigatures 开启合字；支持 ${shellType}、${shellCommand}、${shellPromptInput} 自定义标签。Run Recent Command 显示历史来源文件。
  • 视图移动与导航控制：视图容器可直接通过右键菜单在主侧栏、副侧栏和面板区域间移动。新增 workbench.navigationControl.enabled 隐藏标题栏的前进/后退按钮。
  • 扩展搜索与下载：扩展视图搜索结果中已安装扩展置顶；支持从右键菜单下载扩展 VSIX 文件。
  • 企业扩展管控：新增 extensions.allowed 设置，可基于发布者、扩展 ID、版本、平台配置允许或阻止安装。
