核心亮点

Agent HQ – 统一视图管理所有代理会话

05

MCP 组织级管控

支持通过 GitHub 组织策略配置 MCP 注册表，限制可安装和启动的 MCP 服务器，提升企业安全性。

  • Diff 编辑器可选中复制删除代码：在行内差异视图中，被删除的行现在可以直接选中并复制文本。
  • Go to Line 改进：支持字符偏移导航（:599）、负数行列、越界平滑跳转至首尾。
  • 刷新图标集：codicon 图标重新设计，曲线更柔和、比喻更准确，提升可读性。
  • 多文件差异编辑器跨文件导航：在多文件差异编辑器中可使用上下箭头逐文件浏览变更。
  • 诊断悬浮窗复制按钮：错误、警告等诊断悬浮窗右上角新增复制按钮。
  • 命令调色板口音不敏感过滤：搜索命令时忽略重音符号，适配多语言键盘布局。
  • 高级设置标签：设置编辑器新增 @tag:advanced 筛选，隐藏面向高级用户的复杂配置项。
  • 内联建议可暂停：从沟槽图标可暂停内联建议一段时间，支持选择时长。
Terminal

终端

  • Terminal IntelliSense 默认启用：输入命令时自动弹出补全提示，支持多 shell 和 Git 分支等高级规格。
  • 终端命令附加到聊天上下文：右键命令装饰可将命令、输出及退出码附加为聊天上下文。
  • 终端输出内嵌至聊天（实验性）：通过 chat.tools.terminal.outputLocation 设置，输出可在聊天面板内直接查看，失败时自动展开。
  • 隐藏聊天终端管理：终端标签栏显示隐藏终端数量，可通过快速选择或溢出菜单定位。
  • Shell 特定提示与命令重写：为 PowerShell、bash、zsh、fish 定制描述；PowerShell 中 && 自动重写为 ;。
  • 文件重定向检测（实验性）：通过 chat.tools.terminal.blockDetectedFileWrites 阻止自动批准写入文件的重定向命令。
AI

AI 代理与 MCP

  • Agent Sessions 视图：默认启用，按来源分组显示所有活跃代理会话，支持搜索和单视图模式。
  • Plan Agent：选择 Plan 代理后，通过提问生成详细计划，批准后由 Copilot 执行（支持本地/云端）。
  • Cloud Agents 集成：Copilot coding agent 从 PR 扩展迁移至 Copilot Chat 扩展，支持与 GitHub Mission Control 交互。
  • CLI Agents 集成：可在聊天编辑器或集成终端中创建/恢复 Copilot CLI 会话，发消息、切换模型、附加上下文。
  • 工具审批与信任：外部数据工具支持后置审批；可一次性信任整个 MCP 服务器或扩展所有工具。
  • 保存对话为提示：使用 /savePrompt 将对话保存为可复用的 .prompt.md 文件。
  • MCP 组织级管理：通过组织策略配置注册表，强制使用注册表中的服务器，增强管控。
  • MCP 工作区安装：MCP 服务器可安装至 .vscode/mcp.json，方便团队共享。
  • MCP 认证增强：支持 CIMD OAuth 流程和动态 scope 升级，遵循最小权限原则。
  • 自定义代理元数据：agent.md 文件支持 target 属性指定运行环境（vscode/github-copilot），及 name、handoffs 等属性。
