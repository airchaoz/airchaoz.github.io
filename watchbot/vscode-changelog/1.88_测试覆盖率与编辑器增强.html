Editor

编辑器与工作台

  • 自定义编辑器标签：通过 workbench.editor.customLabels.patterns 配置，使用 ${filename} 等变量区分同名文件。
  • 锁定滚动：命令 View: Toggle Locked Scrolling Across Editors 或设置临时按键绑定 workbench.action.holdLockedScrolling。
  • Activity Bar 移至底部：新增 workbench.activityBar.location 设置 bottom，并改善了顶部时的外观。
  • 搜索编辑器单点操作：设置 search.searchEditor.singleClickBehaviour 可配置单击搜索条目直接打开 Peek Definition。
  • 快速搜索改进：文件名分隔符粘性显示，悬停时显示操作按钮；新增分隔符间导航快捷键（Alt/Cmd+上下）。
  • Minimap 节标题：Minimap 显示折叠标记区域，便于定位 #region 或 MARK: 注释段。
  • 重构预览键绑定：重命名符号预览改为 ⌘Enter/Ctrl+Enter，与代码操作预览一致。
  • 差异编辑器 Staging/Revert 按钮：侧边独立 gutter 支持暂存/撤销修改块，可选中部分操作；可隐藏（diffEditor.renderGutterMenu）。
  • 浮动窗口支持自定义编辑器：将编辑器拖到浮动窗口功能现在支持 webview 编辑器（如 Markdown预览、Hex Editor），跨窗口需重载。
Terminal

终端与测试

  • Shell Integration 在调试终端默认启用：调试终端自动启用 shell integration，提供命令执行跟踪等功能。
  • 运行最近命令改进：Run Recent Command 显示上次执行时间，取消后恢复终端状态。
  • 检测链接改进：打开检测链接前预览结果并高亮来源；去除重复链接，格式一致。
  • 文件路径带空格链接格式支持：新格式 FILE path:line:column 可正确解析含空格路径。
  • 终端粘性滚动支持透明背景：可通过 terminalStickyScroll.background 设置透明色，使滚动时文本穿透。
  • 测试覆盖率 API：原生覆盖率：按钮运行、Ctrl+; Ctrl+Shift+A 运行全部、Ctrl+; Ctrl+Shift+L 运行上次。覆盖信息默认行号叠加，可切换内联显示。
  • 测试消息颜色代码支持：测试失败消息中的终端颜色代码现正确着色，不再显示原始转义字符。
  • 传入文件装饰：合并/变基前，对从远程获取但未合并的文件显示装饰标记，需启用 git.autofetch 和 git.decorations.enabled。
