核心亮点

01

Merge Editor 体验大升级

冲突文件自动显示「在合并编辑器中打开」按钮，结果文件不再修改，冲突标记保持原样但隐藏，复选框可替换为任一侧或 base。

02

全新 Code Action 控件

自定义控件替代简单菜单，可悬停查看禁用原因，支持自定义快捷键导航。

03

终端 Shell 集成与平滑滚动

Fish 和 Git Bash 集成（实验性手动安装），新增平滑滚动设置 terminal.integrated.smoothScrolling。

04

Jupyter 笔记图像粘贴

Markdown 单元格可粘贴截图或图片文件（需启用 ipynb.experimental.pasteImages.enabled 与 editor.experimental.pasteActions.enabled）。

05

F2 重命名选择增强

按 F2 循环选择：文件名、完整名称、扩展名，灵活键盘操作。

Editor

编辑器

  • 粘性滚动正式可用：editor.stickyScroll.enabled 控制当前作用域固定在视口顶部，可设置最大行数 editor.stickyScroll.maxLineCount，默认为 5。
  • IntelliSense 匹配配置：新增 editor.suggest.matchOnWordStartOnly 设置，控制首字符是否必须匹配词首。
  • FFmpeg 编解码器扩展：支持 Vorbis、Flac、H.264、VP8、WAV、MP3、Ogg，提升笔记本和 Webview 中音视频播放能力。
  • 注释过滤与装饰：Comments 视图可按文本和解析/未解析状态过滤；编辑器行槽装饰改用 codicons。
Terminal

终端与调试

  • 下划线样式与颜色：支持 kitty 终端转义序列实现下划线样式和颜色（Windows 暂不支持）。
  • 渲染改进：修复缩放模糊问题；minimum contrast ratio 开启时保留色调；新增 terminal.inactiveSelectionBackground 主题键。
  • tasks 重新连接：watch 任务在窗口重载后自动重连，默认启用，可设 task.reconnection 关闭。
  • 调试多实例警告可选：launch.json 添加 suppressMultipleSessionWarning 选项，允许静默启动多个相同配置的调试会话。
源码管理

源码管理与预览特性

  • 提交后操作记忆：新增 git.rememberPostCommitCommand 设置，记住每个仓库上次执行的提交后操作。
  • Markdown 文件移动时更新链接（实验性）：启用 markdown.experimental.updateLinksOnFileMove.enabled 后，移动/重命名文件自动更新内部链接。
  • Settings Profiles 预览：通过 workbench.experimental.settingsProfiles.enabled 启用，支持 CLI 传参 --profile 和 --profile-temp，可创建临时配置文件。
  • Continue Working On 携带未提交改动：启用 workbench.experimental.editSessions.enabled 和 Settings Sync 后，未提交变更可随仓库迁移到不同环境（Codespaces、本地克隆等）。
