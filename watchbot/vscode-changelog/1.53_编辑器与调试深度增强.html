标签页支持换行显示

新增 workbench.editor.wrapTabs 设置，让标签页在空间不足时自动换行，替代滚动条，提升多文件切换效率。

02

标签页状态装饰可配置

通过 workbench.editor.decorations.colors 和 badges 设置，在标签页上显示 Git 状态、错误/警告标记，一目了然。

03

JavaScript 调试支持条件异常断点与 worker_threads

调试器新增条件异常断点，可在异常抛出时基于变量筛选；并支持 Node.js worker_threads 调试，无需额外配置。

04

Notebook 大纲与面包屑导航

笔记本单元格终于出现在大纲视图和面包屑中，支持按代码/标记单元格过滤，大幅提升长笔记本的导航体验。

05

Markdown 预览自动同步图片变更

当磁盘上的图片文件变化（如外部编辑或分支切换），预览自动刷新，避免手动重载。

06

Emmet 性能飞跃与最新特性支持

HTML/CSS 中 Emmet 展开速度提升最高 20 倍，并支持倒数等新语法，编码加速明显。
