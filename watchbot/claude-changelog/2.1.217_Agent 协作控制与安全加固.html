核心亮点

01

子代理并发上限与嵌套限制

新增 CLAUDE_CODE_MAX_CONCURRENT_SUBAGENTS（默认20）限制并发子代理数，且子代理默认不再允许嵌套（通过 CLAUDE_CODE_MAX_SUBAGENT_SPAWN_DEPTH 开启），防止无限制的 agent fan-out 导致资源过载。

02

--max-budget-usd 全面生效

修复了该预算限制不作用于后台子代理的问题，达到预算上限后拒绝新 spawn 并停止正在运行的后台 agent，确保成本可控。

03

背景会话隔离增强

修复了因未规范化符号链接工作目录导致后台会话可能逃逸 workspace 的安全问题，提升隔离可靠性。

04

MCP 工具输出内存泄漏修复

截断的 MCP 工具输出不再在内存中保留完整未截断版本，消除因大输出导致的持续内存膨胀，改善长时间会话稳定性。

05

Auto-compact 触发修复

修复了 Claude Opus 4.8 on Bedrock 上 auto-compact 永不触发及 /compact 在超过限值时失败的问题，确保上下文压缩功能可用。
