TL;DR

核心亮点

01

自动回退模型确保会话不中断

当主模型不可用时，Claude Code 自动切换到 --fallback-model 继续处理，而非每次请求都报错，大幅提升会话可靠性。

02

代码审查直接应用修复

/code-review --fix 将审查发现直接应用到工作树，/simplify 命令现已调用 /code-review --fix，简化代码优化流程。

03

技能支持禁用特定工具

Skills 和斜杠命令在 frontmatter 中设置 disallowed-tools，可在技能激活时临时移除指定工具，实现更细粒度的安全控制。

04

后台 agent/workflow 等待计时器

后响应计时器新增“Waiting for N background agents/workflows to finish”提示，并累计显示后台任务完成时间，改善并行代理可观测性。

05

Thinking 摘要增强可读性

模型思考摘要现在至少显示 3 秒、渲染为 Markdown、最多 10 行，支持 Ctrl+O 展开全文，让 agent 思维过程更易跟进。

06

修复模型切换导致会话卡死

修复了模型或登录切换后残留 thinking-block 签名导致会话卡住的问题，清理历史并添加重试保护，解决高频阻塞 bug。

Security

安全与权限管理

  • 组织市场插件白名单：新增 pluginSuggestionMarketplaces 管理设置，管理员可对组织市场的插件设置白名单，仅在上下文提示建议中出现。
  • 插件市场移除命令新增作用域：claude plugin marketplace remove 现支持 --scope user|project|local，与 add、install、uninstall 命令保持一致，简化权限管理。
  • MessageDisplay 钩子事件：新增 MessageDisplay hook 事件，允许 hooks 在助手消息显示前转换或隐藏文本内容，增强输出控制。
  • Auto 模式取消 opt-in 确认：自动模式不再需要用户主动同意才能启用，简化入门流程。
