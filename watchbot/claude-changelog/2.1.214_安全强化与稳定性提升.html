核心亮点

Bash 权限系统大修

修复文件描述符重定向绕过、超长命令误判、zsh 变量下标的静默执行，以及 help/man 命令的潜在不安全选项，命令超出10000字符时总是弹窗确认，远程会话的确认框不再提前通过。

新增 EndConversation 工具

Claude 能主动终止高度滥用或越狱尝试的会话，行为与 claude.ai 自 2025 年以来的实现一致。

03

后台会话管理修复

修复背景守护进程 shutdown 时错删后续 socket、空闲背景会话导致进程驻留、完成的背景会话无法通过 claude rm 删除等问题。

04

OpenTelemetry 可观测性增强

日志事件增加 message.uuid、client_request_id、tool_source 属性，新增 CLAUDE_CODE_OTEL_CONTENT_MAX_LENGTH 配置截断阈值，支持消息级别关联与工具来源追踪。

05

长运行时工具心跳信号

长时间运行的 tool call 不再沉默，定期发送进度心跳，避免开发者误以为会话卡死。

06

Windows 大规模兼容性修复

修复 PowerShell 5.1 下权限绕过、重定向写入 UTF-16LE 导致读取异常、Python 脚本非 UTF-8 输入输出崩溃、where.exe 等命令误报错误、Socket is closed 代理代理问题等。

Security

安全与权限

  • Docker 命令权限提示：对 docker（含 Podman shim）携带 --url、--connection、--identity 等守护进程重定向选项时弹出确认，防止未授权远程操作。
  • 目录模式匹配行为变更：单段 dir/ 的 allow 规则（如 Edit(src/)）不再错误地自动批准嵌套 dir/ 目录；hook 的 if: 条件也改为仅匹配 /dir，需全深度匹配时使用 /dir/。
  • file 命令权限强化：使用 -m/--magic-file 或 -f/--files-from 的 file 命令现在需要权限确认，不再自动视为只读操作。
  • Bash 权限检查失败关闭：文件描述符重定向形式解析差异导致绕过权限的 bug 已修复，现在此类操作会正确弹窗。
SHELL

Shell 工具与终端

  • Bash pkill 避免误杀自身：修复 pkill -f 模式意外匹配到 CLI 自身进程导致会话被杀的问题（Linux）。
  • PowerShell 工具多项修复：修复子进程等待 stdin 导致超时、Python 脚本非 UTF-8 输入输出崩溃、错误消息含原始 ANSI 转义序列、where.exe 等命令误报错误、Windows PS5.1 下重定向写入 UTF-16LE 等。
  • 流式 JSON 输出截断优化：对于慢速读取的 SDK/管道消费者，exit 时的 drain 现在根据排队字节数动态调整，而非固定 2s 上限。
后台会话

后台会话与守护进程

  • 守护进程 socket 竞争修复：修复了被移位的后台守护进程在 shutdown 时删除其后继的控制 socket，导致下一个客户端错误地杀死健康的替代守护进程。
  • 空闲背景会话清理修复：通过 ← 或 /background 停放的空闲背景会话不再让守护进程和 worker 进程无限期存活。
  • 背景会话删除问题修复：修复从非 git 目录分发的背景会话无法从 agent 视图中删除、已完成会话在服务闲置后无法用 claude rm 删除等问题。
  • 重新打开已停止背景会话修复：当会话存储中存在不可读目录时，重新打开已停止的背景会话时能正确恢复保存的对话。
