核心亮点

Agent View 预览版

新增全局会话列表视图，通过 `claude agents` 查看所有运行中、阻塞或完成的会话，大幅提升 agent 协作的可视化管理能力。

02

`/goal` 命令提供持续任务执行

设置完成条件后，Claude 跨会话轮次持续工作直到达成，并实时显示耗时/轮次/令牌覆盖层。

03

Hook 支持直接执行与继续恢复

新增 `args: string[]` 字段避免 shell 转义，`continueOnBlock` 配置让 `PostToolUse` 钩子失败后可反馈 Claude 继续当前轮次。

04

MCP 重连与环境变量统一

`/mcp Reconnect` 无需重启即可应用 `.mcp.json` 变更，远程 MCP 服务器临时故障自动重试全面启用，且 MCP stdio 服务器自动注入 `CLAUDE_PROJECT_DIR`。

05

压缩提示保留敏感指令

上下文压缩时模型会保留用户指定的敏感指令，避免关键信息丢失。

06

子代理可观测性提升

子代理 API 请求携带 `x-claude-code-agent-id` 头部，OTEL 链路包含 agent_id 属性，便于追踪。
