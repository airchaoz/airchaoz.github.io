核心亮点

01

动态设置命令

新增 `/config key=value` 语法，可在交互式、`-p` 和 Remote Control 中直接修改任意设置（例如 `/config thinking=false`），无需退出当前会话。

02

沙箱 Apple Events 支持

新增 `sandbox.allowAppleEvents` 选项，允许沙箱化命令在 macOS 上发送 Apple Events，解锁自动化工作流（如 `osascript`）。

03

客户端在线标记文件

新增 `CLAUDE_CLIENT_PRESENCE_FILE` 环境变量，指向标记文件后可抑制设备在线时的移动端推送通知。

04

流式输出逐行展示

长段落流式文本现在逐行出现，不再等待第一个换行，减少阅读等待。

05

API 断线自动重试

思维过程中 API 连接断开会自动重试，不再显示“Connection closed while thinking”。

06

子代理面板智能隐藏

空闲子代理超时30秒后自动隐藏，面板上限5行并显示滚动提示，页脚增加键盘快捷键提示。
