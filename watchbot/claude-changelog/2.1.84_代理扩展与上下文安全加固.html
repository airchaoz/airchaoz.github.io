核心亮点

01

PowerShell 工具 (Windows 预览)

新增 PowerShell 工具作为 Windows 可选预览，agent 可在 Windows 环境中直接执行 PowerShell 命令，弥补 Windows 上 native 工具链的缺失。

02

MCP 上下文防护

MCP 工具描述和服务器指令上限 2KB，防止 OpenAPI 生成的服务无限膨胀上下文，提升 agent 会话稳定性。

03

MCP 配置去重

本地 MCP 配置与 claude.ai 连接器自动去重，本地配置优先，避免冲突与重复初始化。

04

后台 Bash 卡住通知

后台 bash 任务停留在交互提示约 45 秒后弹出通知，减少 agent 无声阻塞，提升可观测性。

05

全局缓存兼容 ToolSearch & MCP

启用 ToolSearch 时全局系统提示缓存仍生效，MCP 用户同样受益，显著提升缓存命中率。
