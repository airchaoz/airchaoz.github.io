核心亮点

01

后台会话支持 `/resume`

通过 `claude --bg` 或 agent view 启动的会话可像交互会话一样通过 `/resume` 恢复，并标记 `bg` 后缀

02

启动挂起从 75s 缩短至 15s 超时

当 API 端点（如 captive portal、防火墙）不可达时，旁路调用 15s 超时，避免无限阻塞

03

模型切换仅作用于当前会话

`/model` 变更只影响当前会话，按 `d` 可设为新建会话的默认模型，避免跨会话干扰

04

MCP 分页响应工具丢失修复

修复了 `tools/list` 返回多页时只取第一页、后续工具被静默丢弃的严重问题

05

终端渐进式字符损坏自愈

极长会话中出现的残留/乱码字形不再需要重绘或 Ctrl+L，系统会自动修复

06

`extra usage` 更名为 `usage credits`

CLI 术语统一为 `/usage-credits`（旧名仍兼容），语义更清晰
