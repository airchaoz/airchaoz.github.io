安全与稳定性

  • 修复 PowerShell 内置 cd 函数绕过沙箱：cd..、cd\、cd~ 和 X: 等内置函数可改变工作目录而不被检测，现已补上。
  • 修复沙箱写入白名单在 git worktrees 中过宽：之前覆盖整个主仓库根目录，现在仅共享 .git 目录，且拒绝 hooks/ 和 config。

  • 修复 find 命令导致 macOS 系统崩溃：Bash 工具中 find 在大目录树时耗尽系统 vnode 表并导致主机崩溃。
