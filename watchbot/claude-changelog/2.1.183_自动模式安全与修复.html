核心亮点

01

自动模式安全加固

未授权时阻止破坏性 Git 命令、非该会话 agent 提交的 amend 以及 Terraform/Pulumi/CDK destroy，防止误操作。

02

模型弃用与自动更新警告

请求的模型若已弃用或自动更新，将在 stderr 显示警告，并覆盖 agent frontmatter 中的设置。

03

修复思考块无声完成

模型仅返回思考块时不再静默终止会话，Claude 将重新提示一次。

04

修复 MCP 认证暴露工具

需要认证的 MCP 服务器在 headless/SDK 模式下不再暴露 auth-stub 工具给模型，避免安全风险。

05

调度任务/webhook 触发行为修正

不再被视为键盘输入，归为任务通知，不能批准操作或设置标题，提升自动化安全性。

06

Windows Terminal TUI 恢复

修复了 Windows Terminal 在高负载子代理下全屏 TUI 错乱的问题。
