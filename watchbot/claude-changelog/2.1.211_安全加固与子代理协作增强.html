核心亮点

01

子代理文本/推理流式输出

新增 --forward-subagent-text 标志和对应环境变量，将子代理文本和思考过程纳入 stream-json 输出，便于外部工具集成与调试。

02

权限预览注入攻击防护

修复工具输入中双向覆盖、零宽字符、视觉相似引号等字符可能篡改审批消息的问题，现在会被自动中和，防止视觉欺骗绕过权限审核。

03

PreToolUse hook ask 决策强制生效

当钩子设置了 'ask' 决策时，即使自动模式下也不会被覆盖，确保安全策略在非沙箱 Bash 中仍能触发用户确认。

04

Always allow 规则跨会话持久化

批准的权限规则现在保存到仓库根目录，在 git worktree 和不同会话间保持一致，避免重复审批。

05

环境变量支持科学记数法与数字分隔符

超时、Token 预算、重试次数等整数环境变量现在可接受 1e6、64_000 等写法，提升可读性和配置灵活性。

06

后台 agent 结果报告改进

Claude 现在能准确报告仍在运行的后台 agent 状态，并等待真实完成而非伪造结果，避免虚假反馈。
