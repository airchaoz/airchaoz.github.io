  • sandbox.filesystem.allowWrite 绝对路径修复：之前需要 // 前缀才能使用绝对路径，现在修复后配置更直观，减少用户困惑。
  • sandbox 启用时依赖缺失警告：当 sandbox.enabled: true 但依赖缺失时，启动时显示可视警告而非静默禁用，防止安全降级。
  • bypassPermissions 模式保护目录修复：修复 .git、.claude 等受保护目录在 bypassPermissions 模式下可写无提示的问题，确保目录保护始终生效。
  • MCP 服务器 deny 权限规则生效：修复 deny 规则未能在发送前从模型上下文中移除被禁用的 MCP 工具，导致模型仍可见并可尝试调用。
  • 沙箱内 git log HEAD 修复：修复 sandboxed Bash 中 'ambiguous argument' 错误以及 stub 文件污染 git status 的问题。
