核心亮点

01

Prompt 缓存失效修复

修复 SDK 中 `query()` 调用导致缓存频繁失效的问题，引入输入 token 成本降低高达 12 倍，大幅减少长期会话的 API 开销。

02

/copy 直接写入文件

新增 `w` 键，可将当前选中的内容直接写入文件，绕过剪贴板，解决 SSH 环境下无图形剪贴板的痛点。

03

/plan 支持即时启动

为 `/plan` 添加可选描述参数（如 `/plan fix the auth bug`），无需二次确认，直接进入计划模式并开始执行，加速 agent 规划流程。

04

新增 Cron 立即停止机制

增加 `CLAUDE_CODE_DISABLE_CRON` 环境变量，允许在会话中随时终止已调度的 cron 作业，提升对后台任务的管控能力。

05

努力等级简化

努力等级缩减为低/中/高（移除 max），使用新符号（○ ◐ ●），并增加 `/effort auto` 快速重置为默认值，降低配置复杂性。

06

并行工具调用容错改进

修复非 Bash 工具（Read/WebFetch/Glob）失败时错误取消同级并行调用的行为，现仅 Bash 错误会级联，增强 agent 在并行任务中的稳定性。

Workspace

会话与工作区管理

  • ExitWorktree 工具：新增 ExitWorktree 工具，用于主动退出由 EnterWorktree 进入的工作树会话，便于在多工作树间切换管理。
  • 命令历史优化：改进向上箭头历史记录，在运行多个并发会话时优先显示当前会话的消息，避免混淆。
  • --continue 修复：修复 --continue 在 --compact 后未从最新恢复点继续的问题，确保会话连续性。
  • /clear 不再杀后台：修复 /clear 会杀死所有后台 agent/bash 任务的问题，现只清理前台任务。
  • Background 任务修复：修复后台 Ctrl+B 查询丢失转录或新对话污染的问题；修复工作树隔离中 Task tool resume 未恢复 cwd、通知缺少 worktreePath/Branch 的 Bug。
工具与权

工具与权限改进

  • Bash 自动批准列表扩展：将 lsof, pgrep, tput, ss, fd, fdfind 加入自动批准白名单，减少只读操作弹窗，提升 agent 执行常见排查命令的速度。
  • Agent Tool 模型参数恢复：恢复 Agent 工具上的 model 参数，允许按调用覆盖模型，方便在特定子任务中切换模型。

  • 权限规则匹配修复：修复通配符规则在 heredoc、嵌入式换行、无参数命令等情况下的匹配失败；修复 deny 规则未应用于所有命令形式等问题。
