核心亮点

01

PreToolUse Hook 新增 defer 决策

headless 会话可在工具调用处暂停等待，通过 `-p --resume` 恢复，实现更灵活的安全审查流程。

02

新增 PermissionDenied Hook

当 Auto Mode 分类器拒绝操作时触发，返回 `{retry: true}` 让模型重试，减少因误拒导致的流程中断。

03

非阻塞 MCP 连接

通过 `MCP_CONNECTION_NONBLOCKING=true` 跳过 MCP 等待，`--mcp-config` 服务连接超时缩短至5秒，大幅提升 `-p` 模式的启动速度。

04

Edit/Read 符号链接安全性修复

权限规则 `Edit(//path/**)` 与 `Read(//path/**)` 现在会检查解析后的目标路径，而非仅校验请求路径，防止绕过。

05

自动压缩防死循环检测

连续3次压缩后上下文立即回满时会停止并报错，避免无谓消耗 API 配额。

06

LSP 僵尸状态自动恢复

LSP 服务器崩溃后，会在下次请求时自动重启，无需重启会话。
