代理与会

代理与会话管理

  • claude agents 新标志：新增 --add-dir、--settings、--mcp-config、--plugin-dir、--permission-mode、--model、--effort、--dangerously-skip-permissions 等标志，用于配置分发后台会话。
  • 后台会话修复：修复 macOS 睡眠/唤醒后会话消失（时钟跳跃检测）、不识别已存在 git worktree、daemon 升级后退出崩溃、Windows 网络驱动器死锁、Ctrl+C 可用。
  • 附件会话 URL 点击修复：后台 worker 的 headless browser shim 不应用于附加会话，点击链接正常。
  • 编辑器变量修复：claude agents 'v to open in editor' 使用 daemon 默认编辑器改为使用 shell 的 $EDITOR/$VISUAL。
  • /web-setup 警告替换：/web-setup 现在在替换现有 GitHub App 连接前发出警告。
模型与优

模型与优化

  • Fast mode 默认 Opus 4.7：Fast mode 现在默认使用 Opus 4.7，设置 CLAUDE_CODE_OPUS_4_6_FAST_MODE_OVERRIDE=1 可回退到 4.6。
  • Reactive compaction 改进：首次摘要尝试从原始请求的溢出大小开始，避免浪费一次近乎全上下文的 retry。
  • MCP_TOOL_TIMEOUT 修复：之前远程 HTTP 和 SSE MCP 服务器的 tool call 被固化为 60 秒，现在正确使用配置的超时。
  • 模型建议移除：从 Usage Policy 拒绝消息中移除了陈旧的 /model claude-sonnet-4-20250514 建议。
